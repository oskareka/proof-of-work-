# proof-of-work-

Ten arkusz zawiera wszystkie witrny które udało mi się zgłosić oraz zablokować za pomocą incydent.cert.pl, safebrowsing zgłoszeń do KAS, rejestratorów domen lub hostingów. Poniższe domeny są zapisane wraz z tokenem, który umożliwiał wejscie na strone oszukańczą, przez cały ten czas kiedy walczyłem z domenami musiałem się posługiwać zmiennym user agent oraz przechodzić i łapać poszczególne przekierowania przez witryny oszukańcze. Twoże ten plik jako potwierdzenie moich umiejętności w thread huntingu oraz jako osobistej statystyki.
